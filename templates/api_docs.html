Документация API

Публичная выдача вакансий внешним сервисам через ключи.

1) Раздел ключей
Управление ключами и фильтрами выполняется тут: /api/keys
Ключ задаёт фильтры выдачи (domains/role/recruiter/company_domain/location_type/salary_min_usd/salary_max_usd/risk_label) и лимиты (requests_per_minute, daily_quota).
2) Публичный endpoint
URL:
GET /api/public/vacancies
Авторизация:
X-API-Key: <token>
Параметры (query):
page — номер страницы (default: 1)
per_page — размер страницы (default: 50, max: 200)
include_blocks — 1 чтобы добавить description/responsibilities/requirements/conditions
include_raw_text — 1 чтобы добавить raw_text
3) Пример запроса
curl -s \
4) Rate limits
Лимиты настраиваются в UI для каждого ключа: requests_per_minute и daily_quota. При превышении вернётся HTTP 429.
Ответ сервера
JSON: { page, per_page, total, items[] }. В items всегда есть summary (берётся из summary_en по умолчанию), а опционально добавляются блоки и/or raw_text.
